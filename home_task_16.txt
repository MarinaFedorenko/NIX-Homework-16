CSV
Дан файл в расширении .csv. Структура файла:


заголовок1,заголовок2,...,заголовокN,
значение1-1,значение1-2,...,значение1-N,
значение2-1,значение2-2,...,значение2-N,
...
значениеN-1,значениеN-2,...,значениеN-N,

Построить из этого файла структуру данных, которая позволяет получить доступ к элементам таблицы 
а) по номеру строки и столбца (строки не включают заголовки) 
б) по номеру строки и тексту заголовка.
А также, позволяет получить список заголовков.


CSV Mapper:
На основе механизма CSV parser, создать программный компонент, который из таблицы csv данных создаст список объектов заданного типа. Инициализацию данного объекта провести при помощи рефлексии, определяя поля, связанные с ячейками таблицы по аннотации.
